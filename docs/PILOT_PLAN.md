# Pilot Plan

Status: not yet executed. Depends on **P-0 → M0.1 → T0.1.3** (pilot cohort recruitment), which is a people/relationship task outside engineering scope.

## Target cohort (AGENTS.md T0.1.3)

- Three initial design-partner teachers.
- One writing-intensive teacher.
- One quantitative or laboratory teacher.
- One elective or world-language teacher.
- Weekly feedback sessions with each.

## Pilot readiness gate

Do not invite pilot teachers until the **P-1 release gate** (T1.10.5) passes:

1. All P-1 milestone acceptance criteria pass.
2. No critical security or privacy issue remains.
3. Rollback procedure documented.
4. Support contact and incident process exist.

## Onboarding

Once T1.1.1 (pilot welcome flow) ships, onboarding should take under 5 minutes per AGENTS.md M1.1 goal. `docs/PILOT_PLAN.md` will be expanded with the actual onboarding script and weekly review cadence once T0.1.3 is complete.
